# Calculadora Simples

Este é um projeto de uma calculadora simples em Python que realiza operações básicas: adição, subtração, multiplicação e divisão.

## Funcionalidades

- Operações: +, -, *, /
- Tratamento de erros para divisão por zero e entradas inválidas.
